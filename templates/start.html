    • About this site

    Incidents of officer-related misconduct are rising in number with every passing year - seeing this, I decided to create this site, with the hope that I could show how easy it would be for this information, which is all already public record, to be organized an accessible by Americans everywhere. This site provides a means by which users can check records of individual officers for incidents of misconduct.



    Disclaimer: This site is ONLY for the purposes of my senior project, and is not meant to be used as an official source of information regarding officer misconduct.
